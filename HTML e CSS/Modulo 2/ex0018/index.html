Titulo 1

O CSS é uma linguagem de estilo usada para definir a aparência visual de uma página web. Com ele, é possível controlar o layout, núcleos, tipografia e diversos outros aspectos do design. O CSS é aplicado aos elementos HTML para transformar uma página simples em algo visualmente atraente e organizado. Ao contrário do HTML, que define o conteúdo, o CSS se concentra na estética e na apresentação do conteúdo.
